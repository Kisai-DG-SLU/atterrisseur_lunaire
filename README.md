# Atterrisseur Lunaire - Eagle-1

Projet de pilote automatique RL pour LunarLander-v3.

## Structure

```
├── src/           # Code source (API, GUI, agents)
├── notebooks/     # Notebooks d'expérimentation
├── models/        # Modèles sauvegardés
├── data/          # Données brutes
├── outputs/       # Vidéos, figures, logs
├── logs/           # TensorBoard
└── tests/         # Tests unitaires
```

## Installation

```bash
pixi install
```

## Usage

```bash
pixi run python -m src.train
```
